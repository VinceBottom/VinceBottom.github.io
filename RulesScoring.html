    • Rules and Scoring FAQ

    • - We will be doing a "Snake" style draft (first player picks last every other round)
    • - The draft will be seven rounds
    • - Each team needs: 1 Setter, 2 Outside Hitters, 1 Opposite Hitter, 2 Blockers, and 1 Libero
    • - Each player will have five minutes to choose. If a player cannot make it to the draft, they can choose to have either Lauren draft their roster or we will autodraft the next best player by position.
    We take a players stat totals for each match, and project it to a four set match. This standardizes each match for each player's contributions. It also avoids rewarding players for playing longer matches (which is not good). We calculate per set scores for each stat and multiply that by four to normalize the data to four sets.
    Raw point calculations:
    Player Action Points
    Kill +1
    Block +1
    Dig +1
    Assist +1/2
    Ace +1
    Perfect Pass +1
    Error -1
    There are two reasons for this:
    • - We normalize the stats to equal a four set match
    • - Our data scraping system is not 100% accurate (but very close)
